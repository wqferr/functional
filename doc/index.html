Module functional

A module for functional programming utils.

An iterable refers to either of:

  • A table with contiguous non-nil values; or
  • An Iterator instance.

Info:

  • Release: 0.8.1
  • Author: William Quelho Ferreira

Iterator.create (t)
Iterate over the given iterable.

If t is a table, create an Iterator instance that returns its values one by one. If it is an iterator, return itself.

Parameters:

  • t iterable the values to be iterated

Iterate over the integers in increments of 1.

Iterate over the n first elements and stop.

Parameters:

Iterator:skip (n)

Iterate over the values, starting at the n+1th one.

Parameters:

Iterator:every (n)
Take 1 element every n. The first element is always taken.

Parameters:

  • n integer one more than the number of skipped elements
Iterator:any (predicate)
Checks if any elements evaluate to true.

Parameters:

  • predicate predicate function to evaluate for each element, defaults to 
    not (value == nil or value == false)
Iterator:all (predicate)
Checks if all elements evaluate to true.

Parameters:

  • predicate predicate function to evaluate for each element, defaults to 
    not (value == nil or value == false)
Iterator:count (predicate)
Counts how many elements evaluate to true.

Parameters:

  • predicate predicate function to evaluate for each element; if nil, then counts all elements.
